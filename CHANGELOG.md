# Changelog

## v1.0.0 (2026-06-17)

### Features
- Drag-and-drop upload of .eml and .msg email files
- LLM extraction of 8 structured order fields
- OpenRouter cloud models and Ollama local models support
- Shipment Requests table with server-side filtering
- Configuration page to switch LLM model without redeployment
- RavenDB Cloud persistence

### CI/CD
- Build and Code Quality Check workflow
- Automated GitHub Release on version tag
